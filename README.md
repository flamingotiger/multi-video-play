# Multi player
유투브 비디오 영상의 url을 입력하면 여러개의 영상을 동시에 시청이 가능합니다.
새로고침을 했을 경우에도 리스트는 남아 있습니다.
단, 웹사이트 로켈데이터를 지우게 될경우 리스트는 지워집니다.
오로지 자신의 컴퓨터에서만 리스트를 저장 및 가져올 수 있습니다.
